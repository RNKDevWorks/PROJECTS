
<h1 align="center">SAFELOG PASSWORD VALIDATOR</h1>
A console-based Password Strength Checker built using Core Java that enforces corporate security policies and provides clear suggestions to users for creating strong passwords.

--- 

### Features
#### Password Validation
* Checks minimum length (at least 8 characters)
  
  * Ensures presence of:
    *  Uppercase letter
    *  Lowercase letter
    *  Digit (0–9)
    *  Special character
      
* Smart Suggestions
  * Provides specific feedback instead of generic errors:
    * Password is too short
    * Missing uppercase letter
    * Missing number
    * Missing special character 

* Retry Mechanism
  * Continuously prompts user until a valid password is entered
* Modular Design
  * Uses a separate validation method for clean and reusable code

---

### Technologies Used
* Java (Core Java)
* OOP Concepts (Encapsulation, Modularity)
* String Manipulation
* Loops (for loop, while loop)
* Scanner (User Input)
